# Política de Privacidade - PoupaCash

A extensão **PoupaCash** foi criada para ajudar os utilizadores a economizar com **cupons e cashback** em sites parceiros.

## Informações coletadas
- Histórico de cliques em sites parceiros (somente para calcular cashback)
- Nenhum dado pessoal sensível é coletado.

## Como usamos os dados
- Para verificar se o site é parceiro
- Para calcular e mostrar cashback no popup

## Compartilhamento de dados
- Nenhum dado é compartilhado com terceiros.

## Segurança
- Todos os dados são armazenados localmente no navegador.
- Não vendemos ou divulgamos informações pessoais.
